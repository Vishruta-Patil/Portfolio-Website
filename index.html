    Sudoku Puzzle

    A sudoku puzzle solver and generator. A responsive Sudoku game in HTML, CSS and JavaScript. Implemented using recursive backtracking algorithm. Player entries that directly conflict with other entries on the board are colored red. The color changes gradually to discourage blind guessing, but you are still made aware of clear mistakes. In the same grid if number is repeated the color of initial entry gradually changes to pink. Player can see the solution of puzzle at any time if it is not getting solved. Player can restart the puzzle at any time and use another board.

    • Created - April 2021
    • technologies used - HTML, css, JavaScript
